## 📋 전체 개요
### 목표: 마이퍼스트웹 채널 재시작 - 첫 영상 업로드 및 홍보
### 소요 시간: 총 5-6시간 (주말 집중)
### 결과물: 유튜브 영상 1개, 홍보 완료, 데이터 수집 시작

사이트 정보:

호스팅: 닷홈 유료 서버  
도메인: 자체 도메인 (구매)  
SSL: HTTPS 적용됨  
비용: 도메인 + 서버 + SSL (연간 약 5-10만원 추정)

📅 **오늘 (11/12 화요일) - Optional 준비 작업**
시간: 저녁 30분  
할 일:

- 스크립트 키워드 메모
  - 인트로: 자기소개, 왜 만들었는지
  - 본론: 사이트 시연, 제작 과정, AI 활용법, 닷홈 호스팅 선택 이유
  - 비용 공개: 도메인 + 서버 + SSL (투명하게)
  - 애드센스: 신청 과정, 현재 상태
  - 아웃트로: 다음 편 예고, 구독 유도

- 고양이 심리테스트 사이트 최종 점검
  - 버그 없는지 확인
  - HTTPS 정상 작동 확인
  - 모바일 반응형 체크
  - 로딩 속도 확인

> 비고: 평일이라 힘들면 생략 가능. 주말에 몰아서 해도 됨.
